About the Project

This is the final project for the 'Data Visualization' module.

We have used two data-sets from Kaggle and combined them to create a visualization. Additionally, we have created other visualizations using these data-sets individually.

About the Developers

Found two datasets on Kaggle and downloaded the CSV files

Used d3.js to parse CSVs and create visualizations

Used only JavaScript (no other language) to process data into useable form

Implemented consistent styling through Bootstrap framework

Dataset #1: US Births by County and State (1985-2015)

The dataset is available in CSV format and consists of 8 columns. Key columns include State (STATEFP), County, Month, Year, countyBirths and stateBirths

(Note that for visualizations we have used data from 2006 to 2015)

See on Kaggle

Dataset #2: Baby Names USA from 1910 to 2021 (SSA)

The dataset is available in CSV format and consists of 5 columns: State (STUSPS), Sex (M/F), Year, Name (first name of the baby) and Count

(Note that for visualizations we have used data from 2006 to 2015)

See on Kaggle
